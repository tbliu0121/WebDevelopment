I 网站制作流程：
    1. 与客户沟通，制定方案
    2. 签订合同
    3. 预付定金 30%左右的总额
    4. 初稿审核（美工设计：原型图 -> Psd效果图）
    5. 前端页面设计，后台功能开发；目前比较流行的是前后端分离。
    6. 测试验收
    7. 上线培训（培训用户如何使用网站）
    8. 后期维护（售后服务：半年到一年左右的时间）

项目搭建
    项目文件夹: shopping
    样式类图片文件夹: images  //存放不经常更换的背景图、小图片之类的。
    样式文件夹: css
    产品类图片文件夹: upload  //产品更换时，需要更新产品图片
    字体类文件夹: fonts
    脚本文件夹: js

    模块化开发：
        有些样式和结构会在很多页面出现，如头部和底部，大部分页面都有，此时可以把这些结构和样式作为一个单独的模块，然后重复使用
        一个典型的应用就是common.css共用样式，如版心宽度、清除浮动等。写好一个样式，其余的页面共用此样式。
        模块化开发具有重复使用、修改方便等特点。

    网站TDK三大标签SEO优化
        SEO(Search Engine Optimizaiton)搜索引擎优化，是一种利用搜索引擎的规则提高网站在有关搜索引擎内自然排名的方式。
        SEO的目的是对网站进行深度优化，从而帮助网站获得免费流浪，进而在搜索引擎上提升网站排名，提高网站知名度。

        页面必须有三个标签来符合SEO优化
        Title / Description / Keyword
            title 网站标题是搜索引擎了解网页入口和对网页主题归属的最佳判断点
                网站名(产品名)
            description
            keywords
